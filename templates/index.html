Refresh MOSO session

The MOSO staging session expires every few hours. Paste a fresh GetRatesOp cURL from your browser's DevTools and we'll extract the auth headers automatically — no JSON editing needed.

How to copy the cURL from DevTools
  1. Open viet18.com/pricing/qm in your browser (make sure you're logged in).
  2. Open DevTools (⌘ ⌥ I on Mac · F12 on Windows).
  3. Click the Network tab and check Preserve log.
  4. Run any quote on the page.
  5. Find the request to /exec/GetRatesOp (filter by "GetRates").
  6. Right-click → Copy → Copy as cURL.
  7. Paste below and click Save session.
We only extract: XSRF, user, X-SDK-Namespace, x-property, Cookie, Referer, Origin, Authorization. Everything else is dropped.
